### 페이지 기획

#### 유저 플로

[유입] 랜딩(인트로)
→ [응시] 11문항 (1문항 1화면, 진행률 표시)
→ [자동 채점] 즉시 처리 (서버 왕복 없이 클라이언트 채점 가능)
→ [결과] 레벨명 + 한 줄 정의 즉시 공개 + 결과 저장
→ [리드 폼] 상세 리포트 열람/발송 조건으로 개인정보 입력
→ [상세 결과] 리포트 발송 및 리드 데이터 조회/추출

#### 랜딩 페이지

**타이틀**

- 당신은 AI를 쓰는 사람인가요, 다루는 사람인가요?

**서브 타이틀**

- 11문항 / 약 3분  / 즉시 결과 확인

**레벨 미리보기**

- 뱃지(탐색자→실행자→설계자→선도자)를 가로로 노출

**버튼**

- 진단 시작하기

#### 문항 화면

- 1문항 1화면 , 상단 진행 바
- 보기 선택 후 자동으로 다음 문항(+뒤로가기 버튼) / 이전, 다음 버튼 추가 여부
- 모바일의 경우 한 화면에 문제와 보기가 모두 보여야함
- 새로 고침 시 응답 복원

#### 결과 화면

- 레벨 뱃지 + 명칭(예 : 당신은 **설계자 (AI Architect)** 단계입니다)
    - 4단계 중 어디인지 보여줘야
- 1~2줄 설명
    
    
    | 레벨 | 명칭(국문) | 명칭(영문) | 한 줄 정의 |
    | --- | --- | --- | --- |
    | 1 | 탐색자 | AI Explorer | AI를 알지만 업무에 거의 활용하지 않거나, 단순 질의응답 수준으로 가끔 활용하는 단계 |
    | 2 | 실행자 | AI Operator | 실무에 정기적으로 활용하며 결과물을 만들어내는 단계 |
    | 3 | 설계자 | AI Architect | 프롬프트를 구조화하고 반복개선하며 업무 프로세스에 통합한 단계 |
    | 4 | 선도자 or선도자 | AI Leader or AI Pioneer | 조직 내 AI 활용을 주도하고 확산시키며 방향을 제시하는 단계 |
- 데이터 수집 후 → 응시자 상위 N% 또는 응시자 중 X%가 이 단계  와 같은 비교 지표 추가
- 블러 또는 잠금으로 강점 분석 / 성장 포인트 / 다음 레벨 가이드 영역을 흐리게 보여주고
- 상세 리포트 리드버튼 유도 버튼

#### 리드 폼

| 필드 | 필수 여부 |
| --- | --- |
| 이름 | 필수 |
| 이메일 | 필수 (리포트 발송처) |
| 회사명 / 직무 | 선택 (B2B 리드 스코어링용, 부담 최소화 위해 선택) |
| 개인정보 수집·이용 동의 | 필수 체크 |
| 마케팅 정보 수신 동의 | 선택 체크 (별도 항목으로 분리 — 법적 요건) |

제출 후 즉시 상세 결과 화면 전환 + 이메일 리포트 자동 발송 또는 이메일 발송 버

#### 상세 결과 화면 구성

- 레벨 재확인 (헤더 카드 — 레벨 배지 + 이름 + 태그라인)
- 나의 AI 리터러시 단계 — 4단계 위치 표시 + 응답 기반 개인화 내러티브
- 역량 진단 차트 — 3축(빈도·깊이 / 프롬프트·자동화 / 확산·주도) SVG 레이더 차트(삼각형) + 축별 점수 카드
- 현재 강점 / 성장 포인트 — 2열 카드
- 다음 레벨 액션 플랜 — 구체적 발전 행동
- Codepresso AI 솔루션 — AI Fluent(AI 역량 평가 플랫폼) + Adaptive Lecture(AI 맞춤 학습) 제품 소개

### 채점 로직

**축별 보기 점수 스케일**

- Q1~Q3 (빈도·깊이): A=1, B=1.5, C=2, D=2.5 (최대 2.5)
- Q4~Q11 (프롬프트·자동화 / 확산·주도): A=1, B=2, C=3, D=4 (최대 4.0)

**축 평균 계산**

- 빈도·깊이 평균 = (Q1+Q2+Q3) / 3
- 프롬프트·자동화 평균 = (Q4+Q5+Q6+Q7+Q8) / 5
- 확산·주도 평균(최종) = (Q9+Q10+Q11) / 3 + 교차반영 보너스

**Q4·5·6·8 교차반영 규칙**

- Q4, Q5, Q6, Q8에서 D(=4) 선택 시 각 +0.2를 확산·주도 평균에 가산 (최대 +0.8)
- Q7의 D는 개인 시스템화로 팀 공유 표현 없음 → 가산 제외

**레벨 판정 로직 (허들 방식)**

| 조건 | 레벨 |
| --- | --- |
| 빈도·깊이 평균 ≤ 1.5 | 1 탐색자 |
| 빈도·깊이 평균 > 1.5 이면서 프롬프트·자동화 < 2.5 | 2 실행자 |
| 빈도·깊이 > 1.5 AND 프롬프트·자동화 ≥ 2.5 AND 확산·주도(최종) < 3.25 | 3 설계자 |
| 빈도·깊이 > 1.5 AND 프롬프트·자동화 ≥ 2.5 AND 확산·주도(최종) ≥ 3.25 | 4 선도자 |
